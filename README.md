## Getting Started

### Prerequisites

- [yarn](https://yarnpkg.com/)
- [node](https://nodejs.org/en/)

### Installation

1. Get a free API Key at [https://example.com](https://example.com)
2. Clone the repo

```sh
$ git clone https://github.com/genzade/webcrawlerhttp
```

3. Install packages

```sh
$ yarn install
```

## Usage

1. Run tests


```sh
$ yarn start https://a-valid.url
```
